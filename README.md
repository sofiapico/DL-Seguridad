# DL Seguridad - Landing page

Landing estática de presentación para DL Seguridad Baires S.R.L.

## Cómo usar

1. Abrí `index.html` en el navegador o levantá un servidor local.
2. Para usar las fotos reales, reemplazá archivos en `assets/` con el **mismo nombre base**.

## Formato recomendado de imágenes

Usá preferentemente `.jpg` o `.webp` (evitar `.svg` para fotos).

Ejemplos:
- `assets/uniforme-logo.jpg`
- `assets/vehiculo-dl-seguridad.jpg`
- `assets/operador-monitoreo-1.jpg`
- `assets/operador-monitoreo-2.jpg`
- `assets/control-acceso.jpg`
- `assets/hero-control.jpg`

> Si no existe `.jpg/.webp`, la página usa el `.svg` como fallback.

## Si las fotos se ven raras (blanco/negro o contrastadas)

1. Verificá que no sean SVG vectorizados o exportados con filtros.
2. Re-exportalas en JPG desde el original.
3. Hacé recarga forzada del navegador: `Ctrl+F5` / `Cmd+Shift+R`.
4. Probá desactivar extensiones de alto contraste del navegador.
